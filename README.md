# CV Portfolio – Arylton John Sant'Anna Santos

Portfólio online desenvolvido com **ASP.NET 8 Web API** + **React + Vite + TypeScript**.

## 🚀 Como rodar

### Backend
```bash
cd backend
dotnet run
```
- API: `http://localhost:5000`
- Swagger: `http://localhost:5000/swagger`

### Frontend
```bash
cd frontend
npm install
npm run dev
```
- App: `http://localhost:5173`

## 🛠️ Tecnologias
- ASP.NET 8 Web API
- React 18 + Vite + TypeScript
- Font Awesome 6 (React)
- Google Fonts (Inter)
- Axios

## ✨ Features
- Layout sidebar escura + conteúdo principal
- Modo escuro/claro com persistência
- Bilíngue PT/EN com transição suave
- Contador de visitas
- Download PDF via window.print()
- Animações fade-in com IntersectionObserver
- Timeline de experiência profissional
- Totalmente responsivo
